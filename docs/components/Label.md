## Props

### `color`

A color of the label. The default value is `success`.

| Type                                                                                                            | Required |
| ----------------------------------------------------------------------------------------------------------------| -------- |
| enum('black', 'dark', 'danger', 'disabled', 'info', 'light', 'muted', 'primary', 'success', 'warning', 'white') | No       |

### `textProps`

Text props. Check [Text documentation](Text.md#props).

| Type   | Required |
| -------| -------- |
| object | No       |
