## Props

### `iconProps`

Icon props. Check [Icon documentation](Icon.md#props).

| Type   | Required |
| -------| -------- |
| object | No       |

### `isChecked`

A state of the checkbox.

| Type    | Required |
| --------| -------- |
| boolean | No       |

### `onChange`

A callback called when the checkbox is pressed.

| Type     | Required |
| ---------| -------- |
| function | Yes      |
