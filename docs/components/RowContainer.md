## Props

### `align`

A vertical alignment of the content. The default value is `center`.

| Type                            | Required |
| --------------------------------| -------- |
| enum('top', 'center', 'bottom') | No       |

### `rnViewProps`

React Native's View props. Check [React Native View documentation](https://reactnative.dev/docs/view#props).

| Type   | Required |
| -------| -------- |
| object | No       |
