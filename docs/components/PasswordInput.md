## Props

Check [React Native TextInput documentation](https://reactnative.dev/docs/textinput#props).

### `isDisabled`

If `true`, the text is not editable and a disabled style is applied. The default value is `false`.

| Type    | Required |
| --------| -------- |
| boolean | No       |
