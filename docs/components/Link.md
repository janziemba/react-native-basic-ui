## Props

### `isDisabled`

If `true`, the link is not pressable and a disabled style is applied. The default value is `false`.

| Type    | Required |
| --------| -------- |
| boolean | No       |

### `onPress`

Called when a single tap gesture is detected.

| Type     | Required |
| ---------| -------- |
| function | Yes      |

### `textProps`

Text props. Check [Text documentation](Text.md#props).

| Type   | Required |
| -------| -------- |
| object | No       |
