## Props

### `size`

A size of the icon. The default value is `24`.

| Type   | Required |
| -------| -------- |
| number | No       |

### `value`

A value of the rating between 0 and 5. The default value is `5`.

| Type   | Required |
| -------| -------- |
| number | Yes       |
