## Props

### `onPress`

Called when a single tap gesture is detected.

| Type     | Required |
| ---------| -------- |
| function | Yes      |

### `scale`

A scale of the component when pressed. The default value is `0.95`.

| Type   | Required |
| -------| -------- |
| number | No       |
